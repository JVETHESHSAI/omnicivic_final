# Deliverables

This folder contains the review-ready files prepared for trainers, panel members, and business stakeholders.

## Files

- `OmniCivic_SDET_Batch4_Technical_Presentation.pptx` - final technical presentation aligned to the SDET handbook and business-unit expectations
- `OmniCivic_Business_Technical_Overview.pptx` - supporting business/technical overview deck

## Source

The presentation deck can be regenerated from:

- `presentation-generator/build-presentation.js`

Run:

```bash
cd presentation-generator
npm install
npm run build
```
